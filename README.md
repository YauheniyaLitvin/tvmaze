# Web service API:

## Get shows:
- URL: /shows?page=:num&limit=:count 
- Example: https://yauheniya-litvin-tvmaze.azurewebsites.net/shows?page=1&limit=10

# Scrapper api

```node scrapper start ``` - start importing shows

```node scrapper restart``` - restart importing shows
